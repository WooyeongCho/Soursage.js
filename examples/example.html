원본 Blue loves Red.. Red likes Blue!!

Replace 방식(rep) Blue loves Red.. Red likes Blue!!

단어 Replace 방식(wordrep) Blue loves Red.. Red likes Blue!!

json 파일 내용 : exampleColors = '{"Red" : "#ff0000", "Blue": "#0000ff", "loves": "#ff9400", "likes": "#ff9400"}'

html 내 script 내용 : var soursage = new Soursage(exampleColors); // Soursage 오브젝트를 생성합니다. soursage.color(document.getElementById("s1"), "rep"); // s1 엘리먼트에 특정 단어마다 example.json에서 가져온 색을 입힙니다. rep 방식. soursage.color(document.getElementById("s2"), "wordrep"); // s2 엘리먼트에 특정 단어마다 example.json에서 가져온 색을 입힙니다. wordrep 방식.
